# Animal Raising Game

This is a simple pet-raising game written in Scala.

The player takes care of a virtual animal by feeding it, playing with it, and letting it sleep. The animal has three status values: Hunger, Happiness, and Energy. The goal is to keep these values balanced and prevent the animal from becoming too hungry, too sad, or too tired.

This project demonstrates basic Scala programming concepts, including:
- Variables
- User input
- Conditional statements
- Loops
- String interpolation

To run the game:

scala AnimalGame.scala
